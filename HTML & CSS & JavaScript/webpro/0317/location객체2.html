location객체 사용 방법

		script proc1 함수 내에서 location.href
		a태그의 href에서 script함수호출 : javascript:proc2()
		a태그의 onclick = 함수호출
		get방식 button의 onclick=location.href


a태그 이용 - href=script함수호출 공지사항목록
a태그 이용 - onclick=script함수호출 공지사항목록
get방식 - button에서 onclick으로 location.href수행


location.replace(url) = 새로운 문서로 대체


location.reload() = 현재 문서를 다시 로드 - 페이지 실행 후 원래 페이지가 다시 로드
